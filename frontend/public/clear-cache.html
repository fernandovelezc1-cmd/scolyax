🧹 Limpiar Cache

Esta página limpia todos los datos en caché del navegador para resolver problemas de visualización.

ℹ️ ¿Cuándo usar esto?

• Si ves errores 500 en el admin
• Si los cambios no se reflejan
• Si el panel no carga correctamente
• Después de actualizar el código

El cache se limpiará y serás redirigido automáticamente después de 2 segundos.
