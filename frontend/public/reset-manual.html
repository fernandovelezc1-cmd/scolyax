🔥 Reset Manual de Logros

ℹ️ Situación: El endpoint de reset tiene limitaciones por RLS (Row Level Security) de Supabase. Usa este método manual para limpiar todos los logros directamente desde la BD.

🔐 Opción 1: Desde Supabase Console (RECOMENDADO)

  1. Abre: https://supabase.com/dashboard
  2. Selecciona proyecto: scolyax-mvp
  3. Ve a: SQL Editor
  4. Click: New Query
  5. Copia y pega este SQL:
    -- Resetear logros DELETE FROM public.user_stats; -- Resetear reseñas/feedback DELETE FROM public.user_feedback;
  6. Click: Run (botón azul)
  7. Verifica que dice ✅ Success - 5 rows deleted (o el número de registros)

✅ Opción 2: Después del Reset

  1. Abre: https://scolyax.vercel.app/debug-storage.html
  2. Click: 🔥 Limpiar TODO localStorage
  3. Confirma
  4. Cierra sesión
  5. Haz login nuevamente
  6. Verifica que aparecen 0 logros ✅

📊 Verificar que funcionó

Después de ejecutar el SQL, verifica:

  1. Abre admin: https://scolyax.vercel.app/admin.html
  2. Click: Usuarios
  3. Verifica que todas las cuentas muestren Racha: 0

🧪 Para Nuevo Testing

Una vez reseteado, puedes:

  1. Hacer logout
  2. Hacer login nuevamente
  3. Completar una tarea para generar nuevo logro
  4. Calificar el logro
  5. Verificar en Admin → Calificaciones que aparezca ✅
